# fraccinganimation

Animated fractal visualization with React and WebGL.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   ```bash
   npm run dev
   ```

## Deploy to GitHub Pages

```bash
npm run build
npm run deploy
```
